# iPhone-style Flutter Homepage

This is a minimal Flutter sample that shows an iPhone-style homepage using `Cupertino` widgets.

Run on macOS with Flutter installed (for iOS simulator/device):

```bash
flutter pub get
flutter run -d ios
```

Files:
- [lib/main.dart](lib/main.dart#L1) — main app UI
- [pubspec.yaml](pubspec.yaml#L1) — dependencies
